我的编程之路

我的编程之路并不是一条很典型的路。

大概是在 2018 年的时候，我初次接触“编程”。那时，我使用的是 Scratch。正是 Scratch 这一简单的模块化工具，让我对编程有了简单的初印象：

有趣、富有创造力、让人愉悦

2020 年，我又接触到了 Python。很幸运，我遇到了一位很好的老师 —— 冯老师。我的思维一向很发散，上课的时候就喜欢问东问西。有些老师比较赶“业绩”，喜欢“扼杀”我所有的 Questions。但是冯老师是个例外 —— 他总是回复我的问题，他喜爱、享受这个过程。

正因为有一位好老师的开导，我喜欢上了一行行花花绿绿的代码、一个个虽黑白却似五彩的输入、输出、一个个鲜红的 Error ……

很快，我拿到了生平第 1 个大奖 —— 蓝桥杯国赛二等奖。我真后悔 Except 没想出来不然就国一了 ……

美好总有结束的那一天。结业测试那一天，我的眼中透着不舍。我为了这次测试做出了一个个奇迹 —— 在没有老师的带领下做出了结业测试的代码、在没有图形的“裸”UI 下画出了奇特的图形 …… 这一切，都是为了那一刻 ……

“就此结束了吗？”“并没有！”

在寂静的一瞬之后，我瞬间醒悟：我的编程路并未终结！我还有很多的路可以走！

我化悲哀为动力，用仅有的 Flask 和 HTML 知识写出了一个中英混用的像样的网站。

C …… ++？

路的尽头是 …… C++！

我从 2021 年开始学习 C++，一路上经历了许多。我最深的一次记忆也是我编程路上最痛苦的回忆 —— 2022 年的 CSP-J 初赛，我的分数为 62.5 分。22 年的 CSP-J 有些难度，所以我估计的分数线在 60 分左右。结果 …… 分数线是 …… 63？？？！！！我最终无缘复赛。

这让我对竞赛有了敬畏。虽然 23 年的 CSP-J 比赛仍然只获得斩获省二等奖，但是，“己所不欲，勿施于人”，我是肯定不会在博客里写竞赛滴 ~

开始吧！

既然你已经读完了我上面所述的所有，那我们是时候开启一段美妙的编程之旅了。以下是攻城狮神秘魔咒每个程序员一生中最重要的句子：

            
#include <iostream>

int main()
{
    std::cout << "Hello World!";

    return 0;
}
当你正式开始学习 C++ 时，务必仔细地把这段代码一字不漏地抄到文本编辑器里。

这将是你接触的第一个 C++ 代码，你可以把它抄到任何文本编辑器里，如果你愿意，也可以抄到纸上。接着，你就可以阅读 C / C++ 的第一个章节，它将会告诉你这段代码的意思，并教会你安装那些神秘的文本编辑器。
